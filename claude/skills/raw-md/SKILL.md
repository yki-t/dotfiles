# raw-md

## タスク

指定されたテキスト（または直前のレスポンス）を、Markdown非対応プラットフォーム向けの簡易フォーマットに変換する。

## 変換ルール

| 要素 | 変換 |
|------|------|
| テーブル（3列以上） | ネストリスト形式（1列目をトップレベル、残りを `- key: value` でネスト） |
| テーブル（2列） | フラットリスト形式（`- key: value`） |
| Bold/Italic（`**`, `*`, `_`） | マーカーを除去し、プレーンテキストにする |
| コードブロック | そのまま維持 |
| 見出し、リンク、画像 | そのまま維持 |

## 出力

変換結果をコードブロックで囲んで出力する（そのままコピー＆ペーストできるようにする）。

## 例

### 3列以上のテーブル

変換前:

```
| サービス | 月額 | リージョン |
|----------|------|-----------|
| EC2      | $100 | us-east-1 |
| RDS      | $200 | ap-northeast-1 |
```

変換後:

```
- EC2
  - 月額: $100
  - リージョン: us-east-1
- RDS
  - 月額: $200
  - リージョン: ap-northeast-1
```

### 2列テーブル

変換前:

```
| 項目 | 値 |
|------|-----|
| 名前 | Alice |
| 年齢 | 30 |
```

変換後:

```
- 名前: Alice
- 年齢: 30
```
